# JavaScript Drum Kit

Welcome to the JavaScript Drum Kit project! This project is a fun and interactive way to create a virtual drum kit using JavaScript. With this drum kit, you can play various drum sounds using your keyboard or by clicking on the drum buttons.

## Features

- **Keyboard Interaction**: Press specific keys on your keyboard to play different drum sounds.
- **Clickable Drums**: Click on drum buttons to produce sounds.
- **Visual Feedback**: Animated effects when drums are played.

## Getting Started

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Edge)
- Basic knowledge of HTML, CSS, and JavaScript

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Berender11/JS-Drum-Kit.git

2. **Navigate to the project directory:**

   cd javascript-drum-kit

3. **Open index.html in your web browser:**

You can simply double-click on index.html or use a local server.

**Usage**
Play Drums Using the Keyboard: Press keys such as 'a', 's', 'd', 'f', 'g', 'h', and 'j' to play different drum sounds.
Click on Drum Buttons: Use your mouse to click on the drum buttons to produce sounds.

**Files**
index.html: The main HTML file.
styles.css: Contains the styles for the drum kit.
script.js: Contains the JavaScript code that powers the drum kit.
